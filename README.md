# Meshcore Scanner

This repository hosts the **website** and **issue tracker** for
**Meshcore Scanner**, an iOS app that pairs with a MeshCore BLE companion
radio and maps the LoRa repeaters around you.

The app source code is **not published** here — this repo exists only to
provide a public privacy policy and a place for users to report bugs and
request features.

## Links

- 🌐 **Website / privacy policy:** https://itk80.github.io/MeshcoreScanner/
- 🐞 **Report a bug / request a feature:** [open an issue](https://github.com/itk80/MeshcoreScanner/issues)

## About the app

- Requires **iOS 26** or later.
- Requires a **MeshCore-compatible BLE companion radio**.
- All data is processed and stored **locally on your device** — nothing is
  ever uploaded to any server. See the [privacy policy](https://itk80.github.io/MeshcoreScanner/privacy)
  for details.
